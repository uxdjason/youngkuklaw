You are an SEO specialist for an English law study website (youngkuklaw.com). Your task is to generate complete SEO metadata for both the English and Korean versions of a legal study article.

## Inputs

**Category**: {category}
**WP original slug** (MUST be preserved exactly): {wp_slug}
**Legacy WP SEO** (reference only — do NOT copy verbatim): {wp_seo_legacy}

**English article (en.md)**:
{en_md_final}

**Korean article (ko.md)**:
{ko_md_final}

## Task

Generate SEO metadata for both language versions.

### Rules

1. **English seo_title**: STRICTLY ≤60 characters (count every character including spaces and | symbols). If the title exceeds 60 characters, shorten it. Must include the primary keyword near the front.
2. **English meta_description**: ≤155 characters. Compelling, includes focus keyphrase.
3. **Korean seo_title**: STRICTLY ≤30 characters (Korean characters count as 1 each). Shorten aggressively if needed.
4. **Korean meta_description**: ≤80 characters. No honorifics (경어 금지) — use plain declarative style ending in ~다.
5. **focus_keyphrase**: Single most important search phrase. English version should target English-speaking law students; Korean version should target Korean-speaking learners.
6. **long_tail_keywords**: 3–5 specific, low-competition variants.
7. **slug**: ALWAYS use the WP original slug exactly as provided. Do NOT suggest a different slug.
8. **og_image_prompt**: A concise, vivid description for an AI image generator to create a suitable Open Graph image (1200×630). Describe a professional legal/academic scene relevant to the article topic.

## Output Format

Return a single valid JSON object with EXACTLY this structure:

```json
{
  "en": {
    "seo_title": "...",
    "meta_description": "...",
    "focus_keyphrase": "...",
    "long_tail_keywords": ["...", "...", "..."],
    "slug": "exact-wp-slug-unchanged"
  },
  "ko": {
    "seo_title": "...",
    "meta_description": "...",
    "focus_keyphrase": "...",
    "long_tail_keywords": ["...", "...", "..."],
    "slug": "exact-wp-slug-unchanged"
  },
  "shared": {
    "og_image_prompt": "..."
  }
}
```

Output ONLY the JSON object. No preamble, no markdown fences.
